# A time series chart component
A time series chart component using vue-chartjs and TypeScript.

## Demo
https://y-ota.github.io/vue-timeserieschart-component/

## Requirements
* vue-cli
* TypeScript
* vue-chartjs
* chartjs-plugin-colorschemes
* chartjs-plugin-zoom

## Setup & Debug
`npm install`

`npm run serve`
